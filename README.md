# Robinhood Kodi Addon

Robinhood is a premium Kodi addon designed to let you watch live sports and television channels easily. It gathers the best sports streams from multiple sources, groups them by match, and shows you in real-time which links are active so you don't waste time clicking on dead links.

---

## 🇬🇧 English User Guide

### Key Features

- **Match Grouping**: If there is a game, all the different channel options for that game are grouped together. You won't see a cluttered list of duplicate channels!
- **Health System (Colors)**: Robinhood tells you the status of each link before you click it:
  - 🟢 **Green**: Working perfectly.
  - 🟡 **Yellow**: Might be unstable.
  - 🔴 **Red**: Currently offline.
  - ⚪ **White**: Untested or new link.
- **Bilingual Support & Simple Layout**: Easy to use menus with zero configuration required.

### Recargar Provider (Reload Button)
Inside each section, you will see a blue option at the top called **[COLOR blue]Recargar Provider[/COLOR]**. If you click it, the addon will instantly clear the temporary memory for that section and reload the page. Use this to get the latest matches or update the schedule!

### Installation

1. Download the `plugin.video.robinhood.zip` file to your device.
2. Open Kodi and go to **Add-ons**.
3. Click the **Box Icon** (Add-on Browser) in the top-left corner.
4. Select **Install from zip file** (make sure "Unknown Sources" is enabled in Kodi settings).
5. Find the downloaded zip file and click **OK**.
6. Wait for the "Add-on installed" notification.

### Requirements
- Kodi 19, 20, or 21.
- `InputStream Adaptive` enabled (usually enabled by default under Add-ons > My Add-ons > VideoPlayer InputStream).

## 🇪🇸 Guía de Usuario en Español

Robinhood es un complemento de Kodi premium diseñado para que puedas ver deportes en vivo y canales de televisión fácilmente. Reúne las mejores transmisiones deportivas de múltiples fuentes, las agrupa por partido y te muestra en tiempo real qué enlaces están activos para que no pierdas tiempo haciendo clic en enlaces caídos.

### Características Clave

- **Agrupación de Partidos**: Si hay un partido, todas las opciones de canales disponibles se agrupan dentro del mismo evento. ¡No verás una lista desordenada de canales duplicados!
- **Indicadores de Estado (Colores)**: Robinhood te dice el estado de cada enlace antes de abrirlo:
  - 🟢 **Verde**: Funcionando perfectamente.
  - 🟡 **Amarillo**: Puede estar inestable.
  - 🔴 **Rojo**: Caído actualmente.
  - ⚪ **Blanco**: Enlace nuevo o sin probar.
- **Diseño Sencillo**: Menús fáciles de usar y sin necesidad de configuraciones complicadas.

### Botón de Recargar
Dentro de cada sección, verás una opción azul arriba del todo llamada **[COLOR blue]Recargar Provider[/COLOR]**. Si haces clic en ella, el complemento borrará la memoria temporal de esa sección y volverá a cargar la lista. ¡Úsalo para obtener los últimos partidos disponibles o actualizar los horarios!

### Instalación

1. Descarga el archivo `plugin.video.robinhood.zip` en tu dispositivo.
2. Abre Kodi y ve a la sección de **Add-ons** (Complementos).
3. Haz clic en el **Icono de la Caja** (Explorador de complementos) arriba a la izquierda.
4. Selecciona **Instalar desde archivo zip** (asegúrate de tener "Orígenes desconocidos" activado en los ajustes de Kodi).
5. Busca el archivo zip descargado y haz clic en **Aceptar**.
6. Espera a que aparezca la notificación de "Add-on instalado".

### Requisitos
- Kodi 19, 20 o 21.
- Tener activado `InputStream Adaptive` (normalmente ya viene activo por defecto en Ajustes > Add-ons > Mis Add-ons > InputStream de VideoPlayer).
